CS203: Programming with Data Structure

Homework 1

Background

This assignment will require that you put together the various parts from the first two lab exercises to build the first model of our road network.

Programming Task

Write a Java program that does the following:

  1. Read in road network information from a text file (road.txt)
  2. Read in cellphone tower information from a text file (tower.txt)
  3. Generate 2D drawing (map.png) of the model conveying the following:
    1. The road network and cellphone towers
    2. The signal strength eminating from each tower
  4. Generate a text report (report.txt) of the model

Format for road network information:

nodeId  x  y

Where:

  • $nodeId$ is a unique location identifier
  • $x$ and $y$ are longitude and latitude respectively

Format for cellphone tower location:

towerId  x  y

Where:

  • $towerId$ is a unique location identifier
  • $x$ and $y$ are longitude and latitude respectively

Analysis Task

There is no analysis report required for this assignment. We will analyze performance and algorithm strategy in future assignments.

Submission Guide

Please submit all your java source code and a runable jar file (hw1.jar). It is highly recommended that you test your java program from the command line as follower:

java -jar hw1.jar road.txt tower.txt map.png report.txt

If everything works as expected, then your program should generate the files as specfied.
