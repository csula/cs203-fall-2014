CS203: Programming with Data Structure

Homework 2

Background

This assignment will require that you put together the various parts from the first two lab exercises to build the first model of our road network.

Programming Task

Write a Java program that does the following:

  1. Read the road network information from a text file (road.txt)
  2. Read the cellphone tower information from a text file (tower.txt)
  3. Generate 2D drawing (map.png) of the model conveying the following:
    1. The road network and cellphone towers
    2. The signal strength eminating from each tower
  4. Generate a text report (report.txt) of the model

Road Network

Format for road network information:

j  x  y

Where:

  • $j$ is a unique node identifier
  • $x$ and $y$ are longitude and latitude respectively

Here is an example of an map file:

1 100.00 200.00
2 101.02 201.22
3 101.50 202.22

Cell Tower Locations

The format for cellphone tower locations:

k x y sigma

Where:

  • $k$ is a unique tower location identifier
  • $x$ and $y$ are longitude and latitude respectively
  • $\sigma$ is the signal strength

Here is an example of a celltower file:

1 100.01 200.04 0.2
2 100.50 200.20 0.1
3 101.10 200.50 0.2

A few things to consider when implementing the code:

  • The drawing library StdDraw.java operates on a coordinate system that $[0,1]$ which means that you will have to scale and adjust corners accordingly.
  • You should rely on Object-Oriented techniques to help out with some of calculations, e.g. consider a Point class that computes and translates its longitude and latitude coordinate to StdDraw $(x,y)$ coordinate.

Report File

Your report file for this homework assignment is simple. Depending on the coverage of your cellphone tower placement, you record either "success" or "drop signal" in your report file.

Analysis Task

There is no analysis report required for this assignment. We will analyze performance and algorithm strategy in future assignments.

Submission Guide

Please submit all your java source code and a runable jar file (hw1.jar). It is highly recommended that you test your java program from the command line as follower:

java -jar hw1.jar road.txt tower.txt map.png report.txt

If everything works as expected, then your program should generate the required files.
